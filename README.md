# شركة الإبداع للإنتاج الفني والتصوير

تطبيق Flutter لشركة إنتاج فني يوفر خدمات التصوير والإنتاج المختلفة.

## المميزات

- واجهة مستخدم عربية بالكامل
- عرض الخدمات المختلفة (تصوير المؤتمرات، إنتاج الفيديو، التصوير الفوتوغرافي)
- نظام حجز الخدمات
- عرض الباقات والأسعار
- نظام التقييمات
- إضافة للمفضلة
- شريط تنقل سفلي

## المتطلبات

- Flutter SDK (3.0.0 أو أحدث)
- Dart SDK
- Android Studio / VS Code

## التثبيت

1. Clone المشروع:
```bash
git clone https://github.com/YOUR_USERNAME/flutter_application_1.git
cd flutter_application_1
```

2. تثبيت المكتبات:
```bash
flutter pub get
```

3. تشغيل التطبيق:
```bash
flutter run
```

## الاختبارات

لتشغيل الاختبارات:
```bash
flutter test
```

## البناء

لبناء التطبيق للأندرويد:
```bash
flutter build apk
```

لبناء التطبيق لـ iOS:
```bash
flutter build ios
```
